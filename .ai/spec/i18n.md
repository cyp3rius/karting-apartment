# i18n Spec

## Locales

| Code | Language | kartingapartment.pl | kartingapartment.eu |
|------|----------|---------------------|---------------------|
| `pl` | Polski | `/` (default, unprefixed) | `/pl/` |
| `en` | English | `/en/` (fallback) | `/` (default, unprefixed) |
| `it` | Italiano | `/it/` | `/it/` |

Prefixed default locale redirects to `/` (301):
- `.pl`: `/pl` → `/`
- `.eu`: `/en` → `/`

## Detection

### kartingapartment.pl
- `/` always serves Polish (no Accept-Language redirect).
- Other languages via `/en/`, `/it/` or the language switcher.

### kartingapartment.eu
1. Cookie `locale` if set to `en|it|pl`.
2. Else `Accept-Language`: `pl*` → `/pl/`, `it*` → `/it/`, else English at `/`.
3. Language switcher updates cookie + `localStorage`.

## Canonical / SEO
- Primary domain: `https://kartingapartment.pl`
- Canonical paths use PL unprefixed (`/`, `/en/`, `/it/`)
- `hreflang` x-default → English (`/en/`)
- OG images: `/og-image-{en,it,pl}.jpg`

## Rules

- Every user-facing string lives in `src/i18n/translations.ts`
- POI names have `name.en`, `name.it`, `name.pl`
- SEO: unique `<title>` and `<meta description>` per locale
- `<html lang="...">` must match active locale
- Host routing lives in `src/middleware.ts` + `src/i18n/config.ts`
