# i18n Spec

## Locales

| Code | Language | URL prefix |
|------|----------|------------|
| `en` | English | `/en/` |
| `it` | Italiano | `/it/` |
| `pl` | Polski | `/pl/` |

Default locale: `en` (no prefix on `/` after redirect).

## Detection (first visit)

1. Check `localStorage.locale` — if set, use it.
2. Parse `navigator.language`:
   - starts with `it` → `it`
   - starts with `pl` → `pl`
   - else → `en`
3. Redirect `/` → `/{locale}/`

User can override via language switcher; choice persisted in `localStorage`.

## Rules

- Every user-facing string lives in `src/i18n/translations.ts`
- POI names have `name.en`, `name.it`, `name.pl`
- SEO: unique `<title>` and `<meta description>` per locale
- `<html lang="...">` must match active locale
